## 📱 About
ZAKA is a professional trading companion that combines a Luno-like interface with AI-powered insights. Track your portfolio, get smart opportunities, and make better trading decisions.

### ✨ Features
- **Live Portfolio Tracking** - Real-time prices and P&L
- **AI Opportunities** - ZAKA finds trades for you
- **Smart Alerts** - Get notified of risks and opportunities
- **Professional Design** - Clean dark theme, mobile-first
- **PWA Ready** - Install on your home screen

## 🎯 How to Use

### On Your Samsung Phone:
1. Open Chrome/Samsung Internet
2. Go to your GitHub Pages URL
3. Tap menu (⋮) → "Add to Home screen"
4. Open ZAKA like a native app!

### Trading Features:
- 📊 View your portfolio with live prices
- 🔥 Check ZAKA's daily opportunities
- 🤖 Get AI analysis on any asset
- 💰 Paper trade to practice
- 📈 Track your performance

## 🛠️ Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
- PWA for mobile installation
- GitHub Pages for hosting

## 📲 Install Instructions

### Samsung Internet:
1. Open ZAKA in Samsung Internet
2. Tap the menu (three lines)
3. Select "Add page to"
4. Choose "Home screen"

### Chrome:
1. Open ZAKA in Chrome
2. Tap the menu (three dots)
3. Select "Add to Home screen"

## 🎨 Design Philosophy
- **Dark theme** for comfortable night trading
- **Card-based UI** for easy scanning
- **Professional colors** (gold accents on dark)
- **Mobile-first** - works perfectly on Samsung phones
- **Smooth animations** for great UX

## 🔒 Privacy
All data is stored locally on your device. No servers, no tracking.
