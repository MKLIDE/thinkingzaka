# ThinkingZaka

ThinkingZaka is a live macro strategy dashboard that tracks the transition between war economies and peace economies.

The platform monitors key geopolitical and financial indicators to determine whether global conditions favor defensive assets or growth assets.

ThinkingZaka transforms macro signals into clear investment alerts and entry opportunities.

Core monitored assets include:

Gold
Bitcoin
Palantir
QQQ
Solana
Major South African banks

Features

• Live market radar
• Macro regime detection
• Entry zone alerts
• Portfolio mode suggestions
• Real-time market movement
• Strategy notifications

The goal is to convert global macro developments into clear portfolio actions.
